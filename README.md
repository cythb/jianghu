# jianghu
一款文字版的江湖游戏——iOS版
试图重现一款以前很流行的江湖聊天室。

现在我想基本上没有人不使用微信和QQ了。然而微信和QQ也好像越来越倾向与熟人交流，而且微信朋友圈和QQ空间充斥着各种生活照，个人状态。一切的一切都太真实而使得聊天缺乏了某一种感觉或者说已经没有新鲜感与神秘感了吧。

这个聊天室试图让我们找回那份悸动。让我们隐藏在一个虚拟的江湖——与现实并行存在,远离城市的喧嚣与冷漠。
